# SpoNS（スポンズ）
スポーツでつながるSNS

## 概要
スポーツ別に、一緒にプレーする仲間や試合の対戦相手を募集することができます。
ユーザー間のフォローやメッセージ（DM）、募集投稿のブックマークも可能です。
